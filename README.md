🤖 Agentic Job Search Assistant

An AI-powered multi-agent system that autonomously discovers, evaluates, and delivers recent AI and LLM job opportunities in the United States.

This project uses CrewAI agents, LLM reasoning (Ollama), and Tavily search to analyze real job postings, filter them based on relevance and experience level, and automatically send a curated HTML job report via email.

The goal of this project is to demonstrate Agentic AI workflows, autonomous research systems, and LLM-driven automation applied to real-world problems.

🚀 Key Features

🤖 Multi-Agent AI System built with CrewAI

🔎 Autonomous job discovery using Tavily search API

🧠 LLM reasoning to evaluate job descriptions

📄 Real job page scraping instead of relying on search snippets

🎯 Smart filtering based on:

Experience level 

AI / LLM / Generative AI relevance

US-based or remote roles
